Do not include "🤖 Generated with Claude Code" or "Co-Authored-By: Claude" in PR descriptions or commit messages.

**Why:** User preference — keep commits and PRs clean without tool attribution.

**How to apply:** Strip all Claude credit footers from both commit messages and PR bodies.
